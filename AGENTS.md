# Codex Agent Instructions — Kukua Website

## Task
Build the complete Kukua nonprofit website by following the spec in `CODEX-PROMPT.md`. This is a full clone of kukua.ca.

## Approach
1. Read `CODEX-PROMPT.md` thoroughly before writing any code
2. Initialize the Next.js project with TypeScript + Tailwind
3. Build reusable components first (Button, SectionLabel, StatCounter, etc.)
4. Build layout components (TopBar, Header, Footer, MobileNav)
5. Build the homepage — this is the most complex page
6. Build all subpages using the reusable components
7. Ensure responsive design works on all breakpoints
8. Add Framer Motion animations

## Important
- All 39 images are pre-downloaded in `/public/images/`
- Follow the color palette and typography exactly as specified
- Every file needs the Freedom_EXE copyright header
- Output must be `next export` compatible (no server-side features)
- Use the existing image files — do NOT reference external URLs
